---
title: Quickstart
description: Get AirGuide running locally in three terminals — frontend, backend, and Prisma Studio.
---

<Info>
  You will need three terminal sessions open simultaneously: one for the frontend, one for the backend server, and one for Prisma Studio.
</Info>

<Steps>
  <Step title="Install frontend dependencies">
    From the project root (`Airguidev1-main`), install the frontend dependencies:

    ```bash
    npm install --legacy-peer-deps
    ```

    This may take a few minutes. Wait for it to complete before moving on.
  </Step>

  <Step title="Install backend dependencies">
    Open a second terminal and navigate to the `server` directory:

    ```bash
    cd server
    npm install --legacy-peer-deps
    ```
  </Step>

  <Step title="Create the backend .env file">
    Inside the `server` directory, create a file named `.env` with the following contents:

    ```bash .env
    # Database - PostgreSQL on Prisma Cloud
    DATABASE_URL="postgres://f7a35276552eb6c4aa3bd70770cb509caeb73a885b11fcbbc269562a4a873e46:sk_isnKgYV7R84c6doSZiLKK@db.prisma.io:5432/postgres?sslmode=require&pool=true"

    # Server
    PORT=3001
    NODE_ENV=development

    # JWT
    JWT_SECRET=airguide-secret-key-app-in-development-2026
    JWT_EXPIRES_IN=1d

    # CORS
    CORS_ORIGIN=http://localhost:5173
    ```

    <Warning>
      Never commit this `.env` file to version control. The `DATABASE_URL` contains credentials for the shared Prisma Cloud PostgreSQL instance.
    </Warning>
  </Step>

  <Step title="Start Prisma Studio">
    Open a third terminal, navigate to the `server` directory, and start Prisma Studio:

    ```bash
    cd server
    npm run prisma:studio
    ```

    Prisma Studio opens a browser-based UI for inspecting and editing database records directly.
  </Step>

  <Step title="Start the backend server">
    In the second terminal (the backend terminal), start the Express server:

    ```bash
    npm run dev
    ```

    The server starts on port `3001`. You should see a confirmation message in the terminal when it is ready.
  </Step>

  <Step title="Start the frontend">
    In the first terminal (the frontend terminal, at the project root), start Vite:

    ```bash
    npm run dev
    ```

    The frontend starts on port `5173`.
  </Step>

  <Step title="Open the app">
    Navigate to [http://localhost:5173](http://localhost:5173) in your browser. You should see the AirGuide campus map.

    <Tip>
      Register a new account to get started. New accounts are created with `estado: pendiente` and require admin approval before accessing protected routes.
    </Tip>
  </Step>
</Steps>

## Environment variable reference

| Variable | Default | Description |
|---|---|---|
| `DATABASE_URL` | — | PostgreSQL connection string for the Prisma Cloud instance. |
| `PORT` | `3001` | Port the Express server listens on. |
| `NODE_ENV` | `development` | Runtime environment. |
| `JWT_SECRET` | — | Secret used to sign and verify JWT tokens. |
| `JWT_EXPIRES_IN` | `1d` | JWT expiry duration (e.g. `1d`, `12h`). |
| `CORS_ORIGIN` | `http://localhost:5173` | Allowed CORS origin for the backend. Must match the frontend URL. |
